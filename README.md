# A.Serious..approach.in.problem.solving.idea
Η ιδέα αφορά ανθρώπους με δυσκολίες βάδισης στα κάτω άκρα. Μία δύσκολη υπόθεση που με έχει προβληματίσει στο τετράγωνο..παίρνω την ιδέα από τους Μεταβαρώνους. ο Μεταβαρώνος έχει πρόβλημα στο δεξί αυτί αλλά η πολεμική του εμπειρία και το βιονικό εξελιγμένης τεχνολογίας αυτί του κάνει το μειωνέκτημα πλεονέκτημα. Απλό!
Τώρα το ίδιο θα πρέπει να επιδιωχθεί στην περίπτωση της δυσκολίας βάδισης-->τα εξωσκελετικά/ρομποτικά χέρια που ενδυνάμει μπορούν να αποκτηθούν(οι τιμές είναι όχι χαμηλές αλλά μέσω ίντερνετ μπορείς να βρείς λύσεις) θα δώσουν βιονική δυνατότητα στα πάνω άκρα!! πρέπει να το οπτικοποιήσει κάποιος για να δεί την ουσία. Για παράδειγμα ένα ολογραφικό VR/AR περιβάλον με απτικούς σένσορες θα σου δίνει υπερ-αίσθηση αφής..αυτό θα ανανεώνει τη σωματική και ψυχολογική πιθανή κατάπτωση τουλάχιστον σε ένα βαθμό.
